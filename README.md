# 💻 Laboratório de Estudos: C# & .NET 8

Repositório dedicado à documentação do meu aprendizado ativo e evolução no ecossistema Microsoft. 

O objetivo deste projeto é aplicar minha base lógica e experiência prévia em desenvolvimento de backends (JavaScript/Node.js) em um ambiente fortemente tipado, explorando a fundo a **Programação Orientada a Objetos (POO)** e a construção de arquiteturas corporativas robustas com **C#**.

## 🎯 Foco de Aprendizado Atual
- Fundamentos da linguagem C# (Tipos, Variáveis, Estruturas de Controle).
- Entrada e saída de dados no Console.
- Manipulação de Strings e formatação.
- **[Em breve]** Aprofundamento em Programação Orientada a Objetos (Classes, Herança, Interfaces).
- **[Em breve]** Estruturas de Dados e Coleções (List, Dictionary).
- **[Em breve]** Introdução a APIs Web com ASP.NET Core 8.

## 🗂️ Estrutura do Projeto

Os exercícios e laboratórios são construídos como aplicações de Console. Para manter a organização, o projeto utiliza um único ponto de entrada (`Program.cs`), que atua como um menu ou orquestrador para executar as diferentes classes de exercícios (ex: `Ex1.cs`, `Ex2.cs`).

## 🚀 Como executar este projeto

Você pode rodar este projeto de duas maneiras:

**Opção 1: Usando o Visual Studio 2026**
1. Clone o repositório: `git clone https://github.com/DevCardosu/EstudosCSharp.git`
2. Dê um duplo clique no arquivo `.sln` para abrir a Solução no Visual Studio.
3. Altere o arquivo `Program.cs` para chamar o exercício desejado.
4. Pressione `Ctrl + F5` para compilar e executar o projeto no terminal.

**Opção 2: Usando o .NET CLI (Terminal)**
1. Clone o repositório e navegue até a pasta do projeto.
2. Execute o comando para rodar o código:
   ```bash
   dotnet run
